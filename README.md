# AI Customer Churn Predictor
An end-to-end Predictive Analytics tool to identify at-risk customers.

## Features
- Real-time probability scoring.
- Interactive Dashboard using Streamlit.
- Machine Learning logic powered by Random Forest.

## How to Run
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run churn_app.py`
